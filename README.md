# 算数指導書デジタルブック v8

C・D・E・Fグレード対応の講師研修用試作版です。

## GitHub Pagesへの更新

ZIPを展開し、リポジトリ直下へ次のファイル・フォルダを上書きアップロードしてください。

- index.html
- app.js
- styles.css
- data
- pdf

アップロード後に **Commit changes** を押し、1～2分待って公開URLを開き直してください。
iPadでは既存のタブを閉じて、新しいタブで開くことを推奨します。

## 主な変更

- C・D・E・Fグレードを収録
- 「すべて・C・D・E・F」のグレード絞り込み
- グレード絞り込みを「すべて・基本・プラス」の上に配置
- ページ数表示から「（3-4表示）」を削除
- ツールバーを横一列にし、「幅に合わせる」が次の行へ落ちないよう変更
- 単元を選ぶと対象グレードのPDFへ自動切替
- 各単元は2ページを縦に表示
