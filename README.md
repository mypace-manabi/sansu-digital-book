# 算数指導書デジタルブック v8

C・D・E・Fグレード対応の講師研修用試作版です。

## GitHub Pagesへの更新

ZIPを展開し、リポジトリ直下へ次のファイル・フォルダを上書きアップロードしてください。

- index.html
- app.js
- styles.css
- data
- pdf

アップロード後に **Commit changes** を押し、1～2分待って公開URLを開き直してください。
iPadでは既存のタブを閉じて、新しいタブで開くことを推奨します。

## 主な変更

- C・D・E・Fグレードを収録

- 各単元は2ページを縦に表示
